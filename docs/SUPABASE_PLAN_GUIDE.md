## 💡 추천 로드맵

### 1단계: 초기 운영 (현재)
- **추천**: **무료 플랜 유지**
- **이유**: 아직 데이터량이 적고, 텍스트 위주의 데이터라 무료 한도로도 충분합니다.
- **팁**: 사진 업로드 시 **WebP 변환 & 압축**을 적용하여 용량을 1/10로 줄이면 1GB로도 오래 쓸 수 있습니다.

### 2단계: 지점 확장 (2호점 이상)
- **추천**: **프로 플랜 업그레이드**
- **이유**:
  1. **데이터 안전**: 여러 지점의 데이터를 책임져야 하므로 **자동 백업**이 필수입니다.
  2. **스토리지**: 지점이 늘어나면 파일 용량(1GB)은 금방 찹니다.
- **수익성**: 지점당 월 이용료를 받게 되면 $25 비용은 충분히 커버 가능합니다.

## ⚠️ 업그레이드 신호 (이럴 때 결제하세요)
1. Supabase 대시보드에서 **"Storage 80% used"** 경고가 뜰 때.
2. 실수로 데이터를 지웠는데 **복구가 절실할 때** (미리 결제해야 백업됨).
3. 2호점 계약이 확정되었을 때.
