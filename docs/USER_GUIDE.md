# 📖 사용자 가이드 - 자라다 ERP

> 아동발달센터 관리자와 치료사를 위한 사용 설명서입니다.

---

## 🚀 시작하기

### 로그인
1. 브라우저에서 앱 주소 접속
2. 이메일/비밀번호 입력
3. **로그인** 버튼 클릭

---

## 📅 일정 관리

### 새 일정 등록
1. 사이드바에서 **일정 관리** 클릭
2. 우측 상단 **새 일정 등록** 버튼 클릭
3. 아동, 치료사, 프로그램, 날짜/시간 선택
4. **일정 등록** 클릭

### 일정 상태 변경
- **예정**: 예약된 수업
- **완료**: 수업 완료 (발달평가 가능)
- **취소**: 수업 취소
- **이월**: 다음으로 이월

### 일정 수정/삭제
1. 캘린더에서 일정 클릭
2. 모달에서 정보 수정
3. **수정 저장** 또는 **삭제** 버튼 클릭

---

## 📊 발달 평가 작성

### 평가 대기 목록 확인
1. 사이드바에서 **발달 평가** 클릭
2. **평가 대기 목록**에서 아동 확인
3. **발달 평가 작성하기** 클릭

### 평가 작성
1. 각 영역별 체크리스트 항목 체크
   - 언어/의사소통
   - 사회/정서
   - 인지/학습
   - 대근육/소근육
   - 자조/적응
2. **종합 소견** 작성 (부모에게 공개됨)
3. **치료사 메모** 작성 (비공개, 내부용)
4. **평가 저장하기** 클릭

---

## 👶 아동 관리

### 아동 등록
1. 사이드바에서 **아동 관리** 클릭
2. **새 아동 등록** 버튼 클릭
3. 이름, 생년월일, 보호자 정보 입력
4. **등록** 클릭

### 아동 정보 수정
1. 아동 목록에서 해당 아동 클릭
2. 정보 수정 후 저장

---

## 💰 수납 관리

### 결제 등록
1. 사이드바에서 **수납 관리** 클릭
2. **새 결제 등록** 버튼 클릭
3. 아동, 금액, 결제수단 선택
4. **저장** 클릭

### 크레딧 확인
- 각 아동 카드에 남은 크레딧(잔액) 표시
- 크레딧 = 결제 금액 - 사용 금액

---

## ✍️ 블로그 관리

### 글 작성
1. 사이드바에서 **블로그 관리** 클릭
2. **새 글 작성** 버튼 클릭
3. 제목, 본문, 커버 이미지 입력
4. **AI 글/이미지 생성** 버튼으로 자동 생성 가능
5. **게시글 발행** 클릭

### AI 자동 발행
- 설정에서 **AI 자동 포스팅** 요일/시간 설정
- 설정된 시간에 자동으로 블로그 글 생성 및 발행

---

## ⚙️ 설정

### 센터 정보 수정
1. 사이드바에서 **설정** 클릭
2. **센터 정보** 탭에서 이름, 주소, 전화번호 수정
3. **저장** 클릭

### 로고 변경
1. 설정 > **브랜딩** 탭
2. 이미지 업로드
3. 저장

---

## 🔔 알림

### 알림 확인
- 우측 상단 🔔 아이콘 클릭
- 새 상담 문의, 일정 변경 등 알림 확인

---

## ❓ 자주 묻는 질문

**Q: 발달 평가가 보이지 않아요**
> 수업 상태를 "완료"로 변경해야 평가 대기 목록에 나타납니다.

**Q: 부모 앱은 어떻게 접속하나요?**
> 아동 상세 페이지에서 "초대 코드 발급" 후 부모에게 전달하세요.

**Q: 블로그 AI가 작동하지 않아요**
> 설정 > GOOGLE_AI_KEY가 올바르게 입력되었는지 확인하세요.
