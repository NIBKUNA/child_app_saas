# 🌐 센터 커스텀 도메인 연결 가이드 (2026.02.13 최종 - 자동화 버전)

> **🚀 업데이트 완료**: 이제 SQL이나 복잡한 설정 필요 없이, **관리자 페이지에서 1분 만에 도메인을 연결**할 수 있습니다.

---

## 1. 사전 준비 (도메인 구입 & 연결 설정)

가비아/후이즈 등에서 도메인을 구매했다면, 아래 설정이 선행되어야 합니다.

### A. Vercel 도메인 추가
1. **Vercel Project Settings** → **Domains**
2. 도메인 입력 (`my-center.com`) → **Add**
3. `www.my-center.com`도 추가 → **Redirect to my-center.com** 설정

### B. DNS 설정 (도메인 구입처)
1. 가비아/후이즈 DNS 설정 접속
2. **A 레코드**: `@` → `76.76.21.21` (Vercel IP 확인)
3. **CNAME 레코드**: `www` → `cname.vercel-dns.com`

---

## 2. 센터 생성 & 도메인 자동 매핑 (핵심 ✨)

이제 앱에서 바로 연결하세요.

1. **슈퍼 어드민 로그인**
2. **[전체 센터 관리]** 페이지 이동
3. 상단의 **[+ 새 지점 개설]** 버튼 클릭
4. 기본 정보 입력 (이름, 영문 슬러그)
5. **👉 [커스텀 도메인 (선택)]** 칸에 도메인 입력
   - 예: `my-center.com` (앞에 http:// 등을 뺀 순수 도메인)
   - *입력하지 않으면 기본 주소(slug)만 사용합니다.*
6. **[지점 생성하기]** 클릭

**🎉 끝입니다!**  
이제 해당 도메인으로 접속하면 자동으로 매핑된 센터 홈페이지가 뜹니다.

---

## 3. 연결 확인 방법

제대로 연결되었는지 보고 싶다면?

1. 연결한 도메인(`my-center.com`)으로 접속
2. **센터 홈페이지(메인)** 가 잘 뜨는지 확인
3. **확실한 검증**: 키보드 `F12`를 눌러 콘솔 창을 엽니다.
   - `✅ [CenterContext] Loaded: 센터이름` 로그가 보이면 ID 매핑까지 완벽하다는 뜻입니다.

---

### ❓ 자주 묻는 질문

**Q. 접속했는데 "센터 정보를 불러오는 중..." 에서 멈췄어요.**
A. 도메인 철자가 틀렸거나, `www` 설정이 꼬였을 수 있습니다.  
   - 브라우저 주소창에 `www`를 빼고 다시 접속해보세요.
   - Admin 센터 수정 페이지에서 도메인 오타가 없는지 확인하세요.

**Q. 기존에 만든 센터에도 도메인을 연결하고 싶어요.**
A. **[전체 센터 관리]** → 해당 센터 카드 클릭 → **[지점 정보 수정]** 버튼 → **커스텀 도메인** 필드에 도메인을 입력하고 저장하면 됩니다.
